## VSCode 플러그인 개발

### Gemini API 활용 아이디어
- [ ] **코드 생성 및 리팩토링:** 자연어 설명 기반 코드 조각 생성, 선택된 코드 리팩토링 (가독성 향상, async/await 변환 등)
- [ ] **인라인 문서 작성기:** 함수/클래스 선택 시 JSDoc, Python docstring 등 자동 생성
- [ ] **스마트 커밋 메시지 생성기:** `git diff` 내용을 기반으로 Conventional Commits 스타일의 커밋 메시지 초안 생성
- [ ] **테스트 케이스 생성기:** 함수/클래스에 대한 단위 테스트 코드(Jest, Pytest 등) 자동 생성
- [ ] **에러 설명 및 해결책 제안:** 빌드/테스트 실패 시 에러 메시지를 분석하고, 원인 설명 및 해결책 제안

### 소스 코드 기반 아이디어
- [ ] **TODO/FIXME 주석 리포터**: 프로젝트 내 모든 `TODO:`, `FIXME:` 주석을 스캔하여 목록으로 보여주고, 클릭 시 해당 코드로 바로 이동하는 기능.
- [ ] **코드 복잡도 분석기**: 선택된 함수나 파일의 순환 복잡도(Cyclomatic Complexity) 등 코드 메트릭을 계산하여 복잡도가 높은 코드를 시각적으로 표시.
- [ ] **의존성 그래프 시각화**: 파일 또는 모듈 간의 `import`/`require` 관계를 분석하여 의존성 그래프를 시각적으로 표시.
- [ ] **로컬 코드 조각 관리자**: 자주 사용하는 코드 조각(snippet)을 키워드로 쉽게 검색하고 삽입하는 기능.
